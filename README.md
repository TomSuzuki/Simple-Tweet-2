## Simple-Tweet-2
- CLIでTweetするためのプログラム  
- Twitterのキーは下の形式でドキュメントフォルダに（移動する場合はUserIDPathを書き換えて）入れておいてください  

temp.json  
~~~~
{  
 "UserID":{
  "ConsumerKey": "**************",  
  "ConsumerSecret": "***********",  
  "AccessToken": "********-*********************",  
  "AccessTokenSecret": "***********************"  
 },
 "UserID2":{
  "ConsumerKey": "**************",  
  "ConsumerSecret": "***********",  
  "AccessToken": "********-*********************",  
  "AccessTokenSecret": "***********************"  
 }
}  
~~~~
